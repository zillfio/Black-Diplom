# SkillTrack

Платформа онлайн-курсов с отслеживанием прогресса пользователя в реальном времени.

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Тестовые аккаунты

|------|-------|--------|

## Структура проекта

```
src/
├── api/           # Работа с данными (mock → API)
│   ├── mock/      # JSON-данные курсов и пользователей
│   ├── auth.ts
│   ├── courses.ts
│   └── progress.ts
├── components/    # UI-компоненты
├── pages/         # Экраны приложения
├── store/         # Zustand-сторы (auth, courses, progress)
├── types/         # TypeScript-типы
└── utils/         # Вспомогательные функции
```

## Функционал

- Просмотр списка курсов с поиском и фильтрами (категория, уровень)
- Страница курса с уроками и прогресс-баром
- Отметка завершённых уроков
- Личный кабинет: текущие и завершённые курсы
- Кеширование прогресса в localStorage
- Роли: ученик и администратор

## Стек

- React 19 + TypeScript
- Vite
- React Router
- Zustand
